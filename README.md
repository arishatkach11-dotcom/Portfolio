# Портфолио танцора | Арина Ткач

[![Сайт](https://img.shields.io/badge/Сайт-онлайн-brightgreen?style=for-the-badge)](https://arishatkach11-dotcom.github.io/Portfolio/) [![GitHub](https://img.shields.io/badge/GitHub-репозиторий-181717?style=for-the-badge&logo=github)](https://github.com/arishatkach11-dotcom/Portfolio)

Одностраничный сайт-портфолио, созданный для начинающего танцора. Проект представляет собой современный, адаптивный лендинг с тёмной темой, который акцентирует внимание на навыках, опыте и визуальном образе танцора.

![Превью сайта](link_to_your_screenshot.png) <!-- Замените на ссылку со скриншотом -->

## 🌟 Особенности

*   **Тёмная эстетика:** Глубокая, драматичная цветовая гамма с акцентами нежного лавандового, создающая атмосферу сцены.
*   **Адаптивный дизайн:** Корректно отображается на всех устройствах — от мобильных телефонов до настольных компьютеров.
*   **Фокус на навыках:** Ключевой раздел, в котором представлены танцевальные направления (Hip-Hop, Girly, House, Jazz Funk) с указанием уровня владения.
*   **Личные данные:** Раздел «Обо мне» с краткой биографией, статистикой (опыт, выступления) и фотографией.
*   **Интеграция с соцсетями:** Прямые ссылки на актуальные профили в Instagram, Telegram и TikTok для быстрой связи.
*   **Чистый код:** Написан на HTML и CSS, легко поддаётся дальнейшей кастомизации.

## 🛠️ Технологии

*   **HTML5** — структура страницы.
*   **CSS3** — стилизация, анимации, адаптивность.
*   **Google Fonts** — шрифты *Playfair Display* и *Quicksand*.
*   **Font Awesome** — иконки для соцсетей и визуальных элементов.

## 🚀 Быстрый старт

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/arishatkach11-dotcom/Portfolio.git
    ```
2.  **Перейдите в папку проекта:**
    ```bash
    cd Portfolio
    ```
3.  **Откройте файл `index.html`** в вашем браузере.

## 🔧 Настройка под себя

Чтобы адаптировать портфолио под свои нужды:

1.  **Замените изображения:** В корневую папку поместите свои файлы и укажите к ним пути в тегах `<img>` внутри `index.html`:
    *   `ava.jpg` — фото в первом экране (Hero).
    *   `uva.jpg` — фото в блоке «Обо мне».
2.  **Измените текст:** В `index.html` отредактируйте имя, описание навыков, текст «Обо мне» и контактный email.
3.  **Обновите ссылки на соцсети:** В блоке `.social-links` замените URL-адреса на свои.
4.  **Настройте цвета:** В корне CSS (секция `:root`) вы можете изменить переменные, чтобы подобрать другую цветовую схему.
